KEY WORKING DIRECTORIES:
- worker/ → Main API server (Cloudflare Workers)
- content/ → Hugo content & pages
- layouts/ → Hugo templates
- assets/ → CSS & styling
- scripts/ → Utilities & automation
- data/ → Database migrations & seeds
- documentation/ → Project docs
- firebase-seeder/ → Firebase setup
- functions/ → Cloud Functions

EXCLUDED (vendor/generated):
- node_modules/ → Dependencies
- .git/ → Git history
- themes/ → Hugo themes
- public/ → Built static site
- _gen/ → Generated files
